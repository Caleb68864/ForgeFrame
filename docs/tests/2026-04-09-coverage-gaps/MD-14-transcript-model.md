# Scenario MD-14: Transcript, TranscriptSegment, WordTiming construction and serialization

## Description
Verify all models in `core/models/transcript.py` -- `WordTiming`,
`TranscriptSegment`, `Transcript` -- construct correctly, that required fields
are enforced, that `confidence` defaults to `1.0`, that the `Transcript`
`asset_id` field is required, and that nested structures round-trip through
JSON and YAML without data loss.

## Preconditions
- Source module exists and is importable: `workshop_video_brain.core.models.transcript`

## Test Cases
- **TestWordTimingRequired**: constructing without `word`, `start`, or `end` raises `ValidationError`
- **TestWordTimingDefaults**: `confidence=1.0`
- **TestWordTimingAllFields**: `word="hello"`, `start=0.5`, `end=1.0`, `confidence=0.95` round-trip through `model_dump()`
- **TestWordTimingConfidenceBoundary**: `confidence=0.0` and `confidence=1.0` both accepted
- **TestWordTimingConfidenceBeyondRange**: `confidence=1.5` accepted (no range validator)
- **TestTranscriptSegmentRequired**: constructing without `start_seconds`, `end_seconds`, or `text` raises `ValidationError`
- **TestTranscriptSegmentDefaults**: `confidence=1.0`, `words=[]`
- **TestTranscriptSegmentWithWords**: list of `WordTiming` objects survives JSON round-trip
- **TestTranscriptSegmentAllFields**: all fields set explicitly; `model_dump()` matches
- **TestTranscriptRequired**: constructing without `asset_id` raises `ValidationError`
- **TestTranscriptDefaults**: `engine=""`, `model=""`, `language=""`, `segments=[]`, `raw_text=""`
- **TestTranscriptIdAutoGenerated**: two `Transcript` instances have different `id` UUIDs
- **TestTranscriptCreatedAtUtc**: `created_at` is timezone-aware UTC
- **TestTranscriptAssetIdRequired**: `asset_id` must be a valid UUID; non-UUID string raises `ValidationError`
- **TestTranscriptAllFields**: all fields set including `segments` with nested `words`; `model_dump()` returns correct nested structure
- **TestTranscriptJsonRoundTrip**: `Transcript.from_json(t.to_json())` produces equal instance
- **TestTranscriptYamlRoundTrip**: `Transcript.from_yaml(t.to_yaml())` produces equal instance
- **TestTranscriptSegmentYamlRoundTrip**: segment-level YAML round-trip preserves nested `words`
- **TestTranscriptEmptySegments**: `segments=[]` in `model_dump()` when no segments added

## Steps
1. Read source module: `workshop_video_brain/core/models/transcript.py`
2. Create `tests/unit/test_transcript_model.py`
3. Implement all test cases
4. Run: `uv run pytest tests/unit/test_transcript_model.py -v`

## Expected Results
- `word`, `start`, `end` required for `WordTiming`
- `start_seconds`, `end_seconds`, `text` required for `TranscriptSegment`
- `asset_id` required for `Transcript`; invalid UUID string raises `ValidationError`
- `confidence=1.0` is the default for both `WordTiming` and `TranscriptSegment`
- `created_at` is timezone-aware UTC
- Nested word lists inside segments survive JSON and YAML round-trips

## Pass / Fail Criteria
- Pass: All construction, validation, and serialization tests pass
- Fail: Any test fails
