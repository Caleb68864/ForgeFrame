# So You Want to Be a Video Editor

This guide is a comprehensive on-ramp to **end-to-end video production**. It walks you through every phase of making a video -- from the initial idea through publishing -- using free, open-source tools centered on [Kdenlive](https://kdenlive.org/).

Rather than dumping a list of buttons and menus on you, this guide teaches a **production pipeline**: a repeatable sequence of decisions and actions that turns raw footage into a finished, published video. You will work through a single beginner project that exercises the entire toolchain, so every concept has a concrete application.

## Who This Guide Is For

You are in the right place if **either or both** of the following describe you:

- **New to video production.** You have not planned, shot, edited, or exported a video before (or you have fumbled through it and want to learn the right way).
- **New to non-linear editors (NLEs) and Kdenlive.** You may have used simple tools like phone editors or screen recorders, but you have never sat down with a full editing application.

### Prerequisites

This guide assumes only:

- Basic computer literacy (navigating files, installing software, using a terminal for simple commands).
- Comfort with file management (creating folders, moving files, understanding file paths).

No prior video, audio, or design experience is required.

## Linux-First, Cross-Platform Friendly

All instructions default to **Linux** (specifically distributions where Kdenlive is readily available via Flatpak, AppImage, or package manager). Where workflows differ on Windows or macOS, you will find clearly marked notes:

> [!tip] Platform notes
> Throughout the guide, look for callouts labeled **Windows** or **macOS** whenever a step differs from the Linux default. If there is no callout, the step works identically on all platforms.

## How This Guide Fits ForgeFrame

This handbook complements the **Workshop Video Brain** workflow tooling in ForgeFrame. Where Workshop Video Brain automates and accelerates parts of the pipeline (project scaffolding, asset tracking, export presets, subtitles), this guide teaches the *concepts* behind those automations so you understand what the tools are doing and why.

You do not need ForgeFrame or Workshop Video Brain to follow this guide. However, if you are using ForgeFrame, the guide will help you understand the decisions the tooling makes on your behalf.
